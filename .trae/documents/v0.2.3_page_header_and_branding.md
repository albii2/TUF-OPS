# v0.2.3 — Page Header System + Branding

## Objective

Create a shared top-of-page pattern for all core pages so the system feels consistent, premium, and scalable. This version will also apply the official TUF Ops branding to the sign-in page to create a professional first impression.

This version will do five things only:
1.  Apply dark mode and logo branding to the sign-in page.
2.  Add a shared page header component.
3.  Add a reusable action grouping component.
4.  Apply the new header pattern to the Dashboard, Organizations, and Opportunities pages.
5.  Standardize top-of-page spacing and action placement.

## Commit Sequence

1.  `feat(auth): apply TUF branding to sign-in page`
2.  `feat(ui): add shared page header component`
3.  `feat(ui): add reusable page action group component`
4.  `refactor(dashboard): adopt shared page header pattern`
5.  `refactor(orgs): adopt shared page header pattern`
6.  `refactor(opps): adopt shared page header pattern`
7.  `chore(ui): harden page-level spacing and consistency`

### **3. `apps/frontend/src/components/ui/page-header.tsx`**

*   **Purpose:** Create one shared page header component for all primary pages.
*   **Create:**
    ```tsx
    import type { ReactNode } from "react";

    export function PageHeader({
      title,
      description,
      actions,
    }: {
      title: string;
      description?: string;
      actions?: ReactNode;
    }) {
      return (
        <div className="flex flex-col gap-4 border-b pb-5 md:flex-row md:items-start md:justify-between">
          <div className="space-y-1">
            <h1 className="text-2xl font-semibold tracking-tight">{title}</h1>
            {description ? (
              <p className="text-sm text-muted-foreground">{description}</p>
            ) : null}
          </div>

          {actions ? <div className="flex items-center gap-2">{actions}</div> : null}
        </div>
      );
    }
    ```

### **4. `apps/frontend/src/components/ui/page-actions.tsx`**

*   **Purpose:** Create a shared action container for header-level actions.
*   **Create:**
    ```tsx
    import type { ReactNode } from "react";

    export function PageActions({ children }: { children: ReactNode }) {
      return <div className="flex flex-wrap items-center gap-2">{children}</div>;
    }
    ```
