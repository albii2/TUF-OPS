# TUF Ops Quick Start Guide (Internal)

## Rep Day-One Workflow (Rep Mission Brief)
1. Open Dashboard → **Rep Mission Brief** and clear Mission Priority first.
2. Check **4-order pace** and **Close This Week**.
3. Work the **Next-Action Execution Queue** from top to bottom.
4. Open each deal in **Deal Command Center**, log follow-up, and advance stage.
5. Request creative/mockups when stage requires assets.
6. End day by checking Cash to Collect and tomorrow follow-ups.

## Director Day-One Workflow (Director Coaching Room)
1. Open Dashboard → **Director Coaching Room**.
2. Review Team 4-order pace and Reps Needing Coaching.
3. Run Coaching Queue: stale deals, stuck deals, near-close pressure.
4. Use Team Opportunities and Territory views for unblock actions.
5. Review personal selling panel and advance own opportunities.

## Owner/Admin Day-One Workflow (Owner Command Center)
1. Open Dashboard → **Owner Command Center**.
2. Execute Mission Priority and Owner Action Queue.
3. Focus on cash blocked, close-this-week value, and territory risk.
4. Move into Territory/Reports/Earnings for deeper analysis.

## Updating Opportunities
- Open **My Opportunities** → select deal.
- Use **Deal Command Center** next action controls.
- Keep follow-up dates current and log notes after every touch.

## Move a Deal Through Close Path
- Use Close Path Timeline + Stage Advancement Drawer.
- Fill only required rep-ready fields for each stage.
- Move quickly (target under 60 seconds per stage update).

## Request Creative / Mockups
- In Deal Command Center, open **Creative Requests**.
- Click **Create Creative Request** and submit design notes + needed items.
- Track active and delivered assets in the creative panel.

## Read 4-Order Pace
- Pace is measured against **4 orders per month minimum**.
- At/above 4 = floor achieved. Under 4 = recovery actions required.

## Understand Lane Penetration
- Lane penetration tracks depth across lanes (Uniform, Team Store, Travel Gear, Letterman).
- Expand lanes inside existing accounts to multiply territory performance.
