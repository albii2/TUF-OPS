# Director Command Mode

## Director role definition
Director is a hybrid seller-manager role: maintains personal pipeline while coaching reps and managing assigned territory.

## Pages used by Director
- /dashboard
- /my-opportunities
- /team-opportunities
- /organizations
- /territory
- /reports (incl team performance)
- /settings

## Visibility model
- Director sees own scoped organizations and operational team surfaces.
- Director can view territory only for assigned zones.
- Director does not get owner-only import/system controls.

## Accountability metrics
- Assigned accounts
- Untouched accounts
- Active opportunities
- Near-close opportunities
- Stuck deals
- Closed won MTD
- Estimated pipeline
- Next coaching note

## Beta testing checklist
1. Verify Director sidebar contains My Opportunities + Team Opportunities + Territory.
2. Verify `/my-opportunities` is role scoped.
3. Verify `/team-opportunities` supports rep/stage/lane/sport/near-close/stuck filters.
4. Verify `/territory` limits zone visibility for Director.
5. Verify Director dashboard includes coaching-focused summaries and weekly coaching module.
