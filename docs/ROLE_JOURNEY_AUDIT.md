# Role Journey Audit (Owner, Director, Rep)

## Scope
- Reviewed route visibility and sidebar mapping in role config.
- Reviewed key workflow pages: dashboard, organizations, opportunities, territory, reports/performance.

## Owner Journey
- Access: full business control surface (all primary pages and admin-style controls).
- Verified controls: can view all organizations/opportunities and use Owner lead import panel.
- Gap fixed: lead import button now provides clear action feedback instead of a dead action.

## Director Journey
- Access: team pipeline, team performance, territory, organizations, and reporting workflows.
- Expected behavior: coach reps, monitor stuck/near-close opportunities, manage territory execution.
- Validation point: role-scoped opportunity listing remains active.

## Rep Journey
- Access: personal pipeline and assigned accounts/opportunities view.
- Expected behavior: execute daily sales workflow and submit creative requests tied to opportunities.
- Implemented: creative request submission inside Opportunity Detail page.

## Completeness Notes
- Navigation structure and page schema were preserved.
- Existing opportunity stage workflow remains unchanged.
- Creative request workflow is now embedded in opportunity execution flow.
