# TUF Lead Import Workflow
1. Owner opens Organizations and uploads CSV in Owner Lead Import.
2. Preview validates required fields and duplicate keys.
3. Import writes organizations as NEW + UNTOUCHED with the CSV `tuf_zone` normalized into territory while rep/director stay Unassigned.
4. Owner uses bulk controls to assign director and rep after the zone is already locked from the lead file.
5. Director reviews assigned schools and sets rep action priorities.
6. Reps execute next actions and create opportunities by lane/sport.
