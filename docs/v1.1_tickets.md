# v1.1 Core CRM Vertical Slice - Ticket Breakdown

This document breaks down the work required for v1.1 of the TUF Ops rebuild.

## Epics

### Epic 1: Organizations Domain

*   **Ticket 1.1:** Create database schema for Organizations.
*   **Ticket 1.2:** Implement CRUD API for Organizations.
*   **Ticket 1.3:** Add ownership and status fields to Organizations.
*   **Ticket 1.4:** Write unit and integration tests for Organizations API.
*   **Ticket 1.5:** Create basic UI for listing and viewing Organizations.

### Epic 2: Opportunities Domain

*   **Ticket 2.1:** Create database schema for Opportunities.
*   **Ticket 2.2:** Implement create/list/detail API for Opportunities.
*   **Ticket 2.3:** Add linkage to Organizations in Opportunities.
*   **Ticket 2.4:** Write unit and integration tests for Opportunities API.
*   **Ticket 2.5:** Create basic UI for listing and viewing Opportunities.

### Epic 3: Workflow and History

*   **Ticket 3.1:** Implement workflow transition guardrails.
*   **Ticket 3.2:** Create stage history/audit trail for Opportunities.
*   **Ticket 3.3:** Write tests for workflow transitions.
