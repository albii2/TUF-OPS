# BETA ROLE USER JOURNEYS

## OWNER
1. Login as owner
2. Open `/users` and create director/rep identities
3. Import leads from `/organizations`
4. Assign territory/director/rep via bulk controls
5. Monitor command-center dashboard and at-risk tiles

## DIRECTOR
1. Login as director
2. Open `/users` to view/manage assigned reps
3. Review `/dashboard` coaching priorities
4. Use `/team-opportunities` and `/team-performance`
5. Review `/territory` pressure and coverage gaps

## REP
1. Login as rep
2. Work `/dashboard` mission queue
3. Execute `/my-opportunities` actions
4. Progress opportunities and close near-close deals

## OPS
1. Login as ops
2. Open `/dashboard` and `/orders`
3. Clear blockers and move production states
