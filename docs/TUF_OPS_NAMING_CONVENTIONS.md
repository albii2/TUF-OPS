# TUF Ops Naming Conventions

## Account Naming
- Use title case for organization names (e.g., Northview Academy, Cedar Hill High, Liberty Prep).
- Account types: School, Youth Program, Club, Rec Team, College, Business/Organization.

## Opportunity / Order Naming
Format:
`[Program Level] [Sport] [Season Code] — [Lane Label]`

Examples:
- Varsity Football FA26 — Uniform
- JV Basketball WI26 — Travel Gear
- 12U Football FA26 — Uniform
- 8th Grade Baseball SP29 — Team Store
- All Athletics FA26 — Letterman

## Season Code System
- SP = Spring
- SU = Summer
- FA = Fall
- WI = Winter

Examples:
- FA26 = Fall 2026
- SP29 = Spring 2029

## Program Levels
School:
Varsity, JV, Freshman, 8th Grade, 7th Grade, Middle School, Elementary, All Athletics

Youth/Rec:
6U through 18U

Club:
National, Regional, Premier, Elite, Select

## Revenue Lanes
- UNIFORM = Uniform
- TRAVEL_GEAR = Travel Gear
- TEAM_STORE = Team Store
- LETTERMAN = Letterman

## Why this matters
Consistent names improve deal handoffs, reporting clarity, pipeline reviews, and faster next-action decisions across sales and operations.
