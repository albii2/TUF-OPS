# Full TUF Ops Tutorial (Internal Operations)

## Operating Doctrine
- 4 orders/month is the non-negotiable floor.
- Lane penetration wins territories.
- Official zones: TUF NORTH, TUF WEST, TUF METRO, TUF SOUTH.

## Rep Mission Brief
Use this daily for mission priority, close-this-week value, cash to collect, stale signals, and execution queue.

## Director Coaching Room
Use this daily to identify reps needing coaching, stuck deals, near-close opportunity value, coverage pressure, and coaching queue actions.

## Owner Command Center
Use this daily for mission priority, cash blocked, close this week, territory risk, and owner action queue. Use Territory/Reports/Earnings for deep analysis.

## My Opportunities
Primary rep workflow table. Open opportunities from here and execute guided next actions.

## Organizations
Account-level ownership and penetration tracking. Prioritize untouched/stale accounts and assign clear next moves.

## Deal Command Center (Opportunity Detail)
Sections:
1. Deal Header (deal/account/sport/lane/zone/rep/value/stage/risk)
2. Close Path Timeline
3. Mission Priority (Issue / Action / Impact)
4. Next Action Console
5. Stage Advancement Drawer rules
6. Creative / Mockup Panel
7. Activity Timeline
8. Outcome Zone (Closed Won / Closed Lost high-consequence)

## Account Penetration Console (Organization Detail)
Use to evaluate account depth, lane coverage, missing lanes, and next relationship move.

## Territory Battlefield
Use to assess zone health, untouched schools/accounts, active opportunities, closed and pipeline revenue, lane penetration by zone, and action-required accounts.

## Earnings Command Center
- Rep: personal revenue, commission, progress, projected earnings.
- Director: personal commission + own override + team production.
- Owner/Admin: total commission liability, net revenue, profitability, override exposure.

## Reports
Use for trend analysis, not daily execution. Dashboards = daily clarity; detail pages = execution.

## Creative Request Workflow
1. Trigger from Deal Command Center.
2. Submit structured request with design notes and needed items.
3. Track request status and delivered assets.
4. Follow up using next action and due dates.

## Daily / Weekly Operating Rhythm
### Daily
- Reps: execute queue + update follow-ups.
- Directors: coach stuck/stale reps and push near-close.
- Owner/Admin: clear blockers and rebalance territory pressure.

### Weekly
- Pipeline scrub by stage.
- 4-order pace review by rep and team.
- Lane penetration review by zone and key accounts.
- Commission/override exposure check.
